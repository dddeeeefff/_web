공지사항
 - 공지글 목록 화면 : notice_list.jsp
 	공지글의 목록 보여주기(페이지 번호에 해당하는 공지글들만 추출)
	공지글 검색(제목, 내용, 제목+내용)
	공지글 링크(공지글 보기 화면으로 특정 공지글을 연결)
	공지글 등록 버튼(공지글 등록 화면으로 연결)
	페이징(공지사항의 게시글들을 페이지로 구분하여 링크)

 - 공지글 보기 화면 : notice_view.jsp
 	받아온 글번호에 해당하는 공지글을 보여줌
	조회수 증가

 - 공지글 등록/수정 화면 : notice_form.jsp
 	공지글에 대한 등록 및 수정을 할 수 있는 폼 제공(수정시 기존 공지글이 나타남)

 - 공지글 등록 처리 : notice_proc_in.jsp - insert, select
 	작성자가 입력한 데이터들을 받아 DB에 저장 후 공지글 보기 화면으로 이동

 - 공지글 수정 처리 : notice_proc_up.jsp - update
 	작성자가 수정한 데이터들을 받아 DB에 저장 후 공지글 보기 화면으로 이동

 - 공지글 삭제 처리 : notice_proc_del.jsp - update
 	작성자가 선택한 공지글을 DB에 삭제 후 공지글 목록 화면으로 이동



자유 게시판
 - 게시글 목록 화면 : free_list.jsp
 	게시글의 목록 보여주기(페이지 번호에 해당하는 게시글들만 추출)
	게시글 검색(제목, 내용, 작성자, 제목+내용)
	게시글 링크(게시글 보기 화면으로 특정 게시글을 연결)
	게시글 등록 버튼(게시글 등록 화면으로 연결- 회원, 비회원 모두 가능)
	페이징(게시사항의 게시글들을 페이지로 구분하여 링크)

 - 게시글 보기 화면 : free_view.jsp
 	받아온 글번호에 해당하는 게시글을 보여줌
	조회수 증가
	게시글 수정 및 삭제 버튼
	 - 회원글일 경우 : 글 작성자에게만 보여줌
	 - 비회원글일 경우 : 비밀번호를 입력받을 화면으로 이동

 - 게시글 등록/수정 화면 : free_form.jsp
 	게시글에 대한 등록 및 수정을 할 수 있는 폼 제공(수정시 기존 게시글이 나타남)
	회원과 비회원의 경우 각각 다른 입력 요소가 생김

 - 게시글 등록 처리 : free_proc_in.jsp - insert, select
 	작성자가 입력한 데이터들을 받아 DB에 저장 후 게시글 보기 화면으로 이동
	회원글은 현재 로그인한 회원 정보를 받아 작성자로 저장

 - 게시글 수정 처리 : free_proc_up.jsp - update
 	작성자가 수정한 데이터들을 받아 DB에 저장 후 게시글 보기 화면으로 이동
	해당 게시글에 댓글이 달렸을 경우 수정 불가

 - 게시글 수정 처리 : free_proc_del.jsp - update
 	작성자가 선택한 게시글을 DB에 삭제 후 게시글 목록 화면으로 이동
	해당 게시글에 댓글이 달렸을 경우 삭제 불가


자유 게시판 댓글(free_view.jsp에서 보여주기)
 - 댓글 목록 보여주기 : free_view.jsp
 	해당 게시글에 속해있는 모든 댓글들을 등록순으로 정렬하여 보여줌
 - 댓글 등록 폼 : free_view.jsp
 	댓글 목록의 위에서 작업

 - 댓글 등록 처리 : free_reply_proc.jsp - insert, update
 	작성자가 입력한 데이터들을 받아 DB에 저장 후 게시글 보기 화면으로 이동
	비회원 등록일 경우 작성자와 비밀번호(삭제시 필요)도 받아야 함

 - 댓글 삭제 처리 : free_reply_proc.jsp - update, update
 	작성자가 선택한 댓글을 삭제(bf_isdel 컬럼 변경) 후 게시글 보기 화면으로 이동
	비회원 댓글 삭제일 경우 비밀번호 입력 폼으로 먼저 이동
