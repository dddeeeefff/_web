jQuery에서의 HTML, CSS 속성 설정

1. 공용 메소드
 - 태그의 종류에 상관없이 사용할 수 있는 공용 메소드
 - text() : 선택된 요소에 텍스트 내용을 설정하거나 리턴받는 메소드(태그는 실행되지 않고 그냥 출력됨)
	문법 : var 변수 = $(선택자).text()	-> 선택된 요소의 텍스트 내용을 리턴받아 변수에 저장(태그는 제외)
		  $(선택자).text("문자열")		-> 선택된 요소에 지정된 문자열을 출력(문자열에 태그사용불가)

 - html() : 선택된 요소에 HTML 내용을 설정하거나 리턴받는 메소드(내용에 태그가 있으면 실행됨 : innerHTML과 동일)
	문법 : var 변수 = $(선택자).html()	-> 선택된 요소의 HTML 내용을 리턴받아 변수에 저장
		  $(선택자).html("문자열")		-> 선택된 요소에 지정된 문자열을 HTML로 출력(태그사용가능)

 - val() : 선택된 컨트롤의 value 속성을 설정하거나 리턴받는 메소드
	문법 : var 변수 = $(선택자).val()	-> 선택된 요소의 value 속성 내용을 리턴받아 변수에 저장
		  $(선택자).val("문자열")		-> 선택된 요소의 value 속성 내용을 지정한 문자열로 변경

 - attr() : 선택된 요소의 지정된 속성 값을 설정하거나 리턴받는 메소드
	문법 : var 변수 = $(선택자).attr(속성명)	-> 선택된 요소의 지정된 속성 값을 리턴받아 변수에 저장
		  $(선택자).attr(속성명, 속성값)		-> 선택된 요소의 지정된 속성 값을 지정한 값으로 변경


2. Box 모델 관련 메소드 : div, table, p 등의 박스 형태 또는 브라우저 화면에 관련된 메소드들
 - height() : 선택된 요소의 높이를 설정 또는 리턴하는 메소드
 - width() : 선택된 요소의 넓이를 설정 또는 리턴하는 메소드
 - innerHeight() : 선택된 요소의 높이(padding 포함, border와 margin 제외)를 설정 또는 리턴하는 메소드
 - innerWidth() : 선택된 요소의 넓이(padding 포함, border와 margin 제외)를 설정 또는 리턴하는 메소드
 - outerHeight() : 선택된 요소의 높이(padding과 border 포함, margin 제외)를 설정 또는 리턴하는 메소드
 - outerWidth() : 선택된 요소의 넓이(padding과 border 포함, margin 제외)를 설정 또는 리턴하는 메소드


3. 위치 관련 메소드
 - offset() : 선택된 요소의 좌료를 설정 및 리턴하는 메소드
	offset() : 요소의 위치객체 리턴(X좌표 : 객체.left, Y좌표 : 객체.top)
	offset({ left:X좌표값, top:Y좌표값 }) : 요소의 위치를 지정
 - offsetParent() : 최초로 위치하는 부모 요소를 리턴
 - position() : 요소의 위치(부모 요소에서의 상대적인 위치)를 리턴
 -scrollLeft() : 선택한 요소의 가로 스크롤 막대 위치를 설정 및 리턴
 -scrollTop() : 선택한 요소의 세로 스크롤 막대 위치를 설정 및 리턴


4. CSS 속성 관련 메소드
 - css() : 선택한 요소에 대해 하나 이상의 CSS속성을 설정하거나 리턴받는 메소드
 - 문법 : var 변수 = $(선택자).css(속성명)		-> 선택된 요소의 지정된 css속성의 값을 리턴받아 변수에 저장
		 $(선택자).css(속성명, 값)			-> 선택된 요소의 지정된 css속성의 값을 지정된 값으로 변경
		 여러 css속성 설정 : $(선택자).css(속성명1:값1, 속성명2:값2, ... , 속성명n:값n);


5. 요소 및 컨텐츠 조작 메소드
 - addClass(클래스명) : 선택한 요소에 하나 이상의 클래스를 추가
 - after(문자열) : 선택한 요소 뒤에 지정한 문자열 삽입
 - append(문자열) : 선택한 요소의 끝에 지정한 내용(태그 사용가능) 추가
 - appendTo(HTML요소) : 선택한 요소를 지정한 요소의 뒤로 옮기라는 명령
 - before(문자열) : 선택한 요소 앞에 지정한 문자열 삽입
 - clone(논리값) : 선택한 요소의 복사본을 삽입
 - insertAfter(HTML요소) : 선택한 요소 뒤에 HTML요소 삽입
 - insertBefore(HTML요소) : 선택한 요소 앞에 HTML요소 삽입
 - prepend(문자열) : 선택한 요소의 시작 부분에 지정한 내용(태그 사용가능) 추가
 - prependTo(HTML요소) : 선택한 요소를 지정한 요소의 앞으로 옮기라는 명령

 - append()와 prepend() 메소드에는 매개변수를 무제한으로 입력할 수 있음
	예) append("a", "b", "c", "d", "e", ...);
 - addClass(클래스명 or 함수명())
	클래스명 : 문자열로 된 클래스 이름이며 띄어쓰기로 구분하면 여러 개의 클래스를 지정할 수도 있음
	함수명() : 클래스명을 만들어 낼 함수로 연산 후 클래스명을 리턴받음
 - clone(논리값)
	자식노드, 텍스트 및 특성을 포함하여 선택된 요소의 복사본을 생성
	true : 이벤트 핸들러도 복사
	false : 기본값으로 이벤트 핸들러는 복사하지 않음


6. 요소 및 컨텐츠 삭제 메소드
 - detach(클래스명 or 아이디명) : 선택한 요소를 제거(데이터및 이벤트는 유지)
 - empty() : 선택한 요소로 부터 모든 자식 노드와 내용을 제거(선택한 요소의 내용만 삭제, 요소는 유지)
 - remove(클래스명 or 아이디명) : 선택한 요소를 제거(데이터 및 이벤트 포함)
 - removeClass(클래스명) : 선택한 요소에서 지정한 클래스 제거
 - removeAttr(속성명) : 선택한 요소에서 지정한 속성 제거(여러 속성 제거시 띄어쓰기로 구분)
 - removeProp() : prop() 메소드로 설정된 속성을 제거
 - toggleClass(클래스명) : 선택한 요소에서 하나 이상의 클래스를 추가 및 제거하는 기능 토글
 - unwrap() : 선택한 요소의 부모 요소를 제거

 - detach() 와 remove() 모두 여러 클래스나 아이디를 삭제하려면 쉼표로 구분하여 입력하면 됨


7. 요소 및 컨텐츠 변경 메소드
 - replaceAll() : 선택한 요소를 새 html요소로 변경
	문법 : $(변경할 내용).replaceAll(변경될 선택자);		변경할 내용이 앞에 들어가고, 뒤에 변경할 선택자를 입력함
 - replaceWith() : replaceAll() 과 동일하며, 매개변수의 순서만 다름
	문법 : $(변경될 선택자).replaceAll(변경할 내용);
 - hasClass() : 선택한 요소에 지정된 클래스 이름이 있는지 확인
	문법 : $(선택자).hasClass(클래스 이름);


8. 래핑 메소드
 - wrap() : 선택한 요소의 주위를 지정한 요소로 래핑-여러 개일 경우 각각 따로 래핑
	문법 : $(선택자).wrap(선택한 요소를 감쌀 요소명);
 - wrapAll() : 선택한 요소의 주위를 지정한 요소로 래핑-여러 개일 경우 모두 묶어서 래핑
	문법 : $(선택자).wrapAll(선택한 요소를 감쌀 요소명);
 - wrapInner() : 선택한 요소의 주위를 지정한 요소로 래핑
	문법 : $(선택자).wrapInner(선택한 요소를 감쌀 요소명);
 - unwrap() : wrap()이나 wrapAll()로 감싼것을 원상태로 돌림
	문법 : $(선택자).unwrap();


9. 요소 탐색 메소드
 - HTML요소를 다른 요소와의 관계를 기반으로 찾기 또는 선택하는데 사용되는 메소드
 - parent([filter]) : 선택된 요소의 직접적인 부모 요소를 리턴
	filter : 여러 부모 요소(들)를 지정하기 위한 선택자로 쉼표로 구분하여 열 개를 입력할 수 있음
 - parents([filter]) : 선택된 요소의 모든 조상 요소를 리턴
	filter : 여러 부모 요소(들)를 지정하기 위한 선택자로 쉼표로 구분하여 열 개를 입력할 수 있음
 - parentUntil(top, filter) : 2개의 매개변수 사이의 모든 조상 요소를 리턴
	top : 선택한 요소부터 top으로 지정된 요소까지를 선택
	filter : 여러 부모 요소(들)를 지정하기 위한 선택자로 쉼표로 구분하여 열 개를 입력할 수 있음
 - closest() : 선택된 요소의 첫번째 조상을 리턴
