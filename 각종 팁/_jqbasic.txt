제이쿼리(jQuery)

제이쿼리 사용법
1. 직접 다운로드 받아 사용
 - https://jquery.com/download/ 에서 최신 버전의 jQuery를 다운 받음
	다운시 production과 development 두 가지 버전이 있는데 아무거나 받아도 상관없음
	production : 압축된 버전으로 jQuery의 사용만을 위한 버전
	development : 개발자용으로 디버깅이 가능한 버전
 - 다운 받은 .js 파일을 jQuery를 사용하는 html 파일에 링크로 연결하여 사용함
	<script src="경로 및 파일명"></script>
	예) <script src="jquery-3.6.0.js"></script>

2. 외부 사이트에 있는 jquery파일을 연결하여 사용
 - google이나 microsoft 등의 사이트에 있는 jQuery 파일을 연결하여 사용할 수도 있음
	단, 해당 사이트에 문제 발생시 jQuery의 사용이 어려워질 수도 있으므로 권장하지는 않음
 - <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.2.0/jquery.min.js"></script>
 - <script src="https://ajax.aspnetcdn.com/ajax/jQuery/jquery-3.2.0.min.js"></script>


제이쿼리란
 - HTML요소를 선택하여 그 요소에 특정 'action'을 수행시킴
 - jQuery 구문은 HTML요소를 선택하고, 그 요소에 대해 몇가지 작업을 수행하기 위해 만들어짐
 - 제이쿼리 구조
	$(선택자).기능(작업할 내용);
	$ : jQuery를 의미하는 기호로 'jquery'를 직접 입력해도 되며, jquery 기능을 사용하겠다는 의미
	선택자 : 기능을 부여할 HTML요소를 지정하는 역할(CSS에서의 선택자와 동일함)
	기능() : 선택된 요소가 행해야 하는 동작을 명시하는 곳
 - 일반적으로 jQuery 소스는 $(document).ready(function() { 작업 공간 });
	현재 문서(document)가 로드된 후(ready) 작업을 시작하라는 의미
	 - 페이지가 로드되기 전에 jQuery가 실행되는 것을 막음


jQuery 선택자
 - CSS의 선택자와 비슷하며, 기능을 부여할 HTML요소를 선택하는 데 사용됨
 - CSS와 마찬가지로 선택자 입력시 띄어쓰기(소속), 쉼표(여러 개) 등을 사용할 수도 있음

1. 태그 선택자 : 선택자에 의해 태그가 직접 선택되는 선택자
 - $("태그명") : 지정한 태그(들)을 선택
 - $("*") : 문서내의 모든 태그들을 선택
 - $(this) : 자기 자신을 의미
 - $("#아이디") : 지정한 아이디를 가진 요소를 선택
 - $(".클래스") : 지정한 클래스를 가진 요소(들)을 선택

2. 그룹 선택자 : 태그 선택자가 하나의 요소를 선택하는 것이면 그룹은 여러 요소를 선택하는 것을 의미
 - 태그 선택자들을 쉼표로 연력하여 여러 요소를 한 번에 선택하게 해줌

3. 노드 선택자
 - 문서의 구조를 이용하여 소속되어 있는 요소들을 선택하는 방법

(1) 하위 선택자
 - 현재 요소에 속해있는 모든 요소들을 의미함
 - 문법 : $("요소A 요소B") -> '요소A' 안에 있는 모든 '요소B'를 의미
	예) $("div a") -> div태그안에 있는 모든 a태그들

(2) 자식 선택자
 - 현재 요소에 속해있는 모든 요소들 중 현재 요소의 바로 아래에 있는 요소들만을 의미함
 - 문법 : $("요소A > 요소B") -> '요소A' 바로 아래에 있는 모든 '요소B'를 의미

(3) 형제 선택자
 - 현재 요소와 동일한 레벨의 요소들을 의미
 - 문법 : $("요소A + 요소B") -> '요소A' 와 동일한 레벨의 모든 '요소B'를 의미

(4) 순서 선택자
 - 첫번째나 마지막 요소를 선택할 때 사용
 4-1) first 선택자
	- 동일한 요소들 중 첫번째로 나오는 요소만을 선택
	- 문법 : $("요소명:first") -> 해당 요소들 중 첫번째 요소만 선택
	 예) $("p:first") -> p태그들 중 첫번째 p태그만 선택
 4-2) first-child 선택자
	- 부모요소를 기준으로 자식 요소들 중 첫번째 요소만 선택
	- 문법 : $("요소명:first-child") -> 해당 요소의 부모 요소를 기준으로 첫번째 요소만 선택
 4-3) last 선택자
	- 동일한 요소들 중 마지막으로 나오는 요소만을 선택
	- 문법 : $("요소명:last") -> 해당 요소들 중 마지막 요소만 선택
	 예) $("p:last") -> p태그들 중 마지막 p태그만 선택
 4-4) last-child 선택자
	- 부모요소를 기준으로 자식 요소들 중 마지막 요소만 선택
	- 문법 : $("요소명:last-child") -> 해당 요소의 부모 요소를 기준으로 마지막 요소만 선택

(5) 폼서식 선택자
 - 폼안에서 사용되는 컨트롤 관련 선택자
 5-1) 입력 요소 선택자
	- input 태그의 type속성 값에 따른 선택자를 의미
	- $(":input")	: input요소를 의미
	- $(":text")	: type="text" 인 요소를 의미
	- $(":password"): type="password" 인 요소를 의미
	- $(":file")	: type="file" 인 요소를 의미
	- $(":radio")	: type="radio" 인 요소를 의미
	- $(":checkbox"): type="checkbox" 인 요소를 의미
	- $(":button")	: type="button" 인 요소를 의미
	- $(":submit")	: type="submit" 인 요소를 의미
	- $(":reset")	: type="reset" 인 요소를 의미
	- $(":image")	: type="image" 인 요소를 의미

 5-2) 상태 선택자
	- 컨트롤들의 상태에 따라 요소를 선택하게 해줌
	- $(":enabled")	: 모든 활성화된 폼 요소들을 선택
	- $(":disabled"): 모든 비활성화된 폼 요소들을 선택
	- $(":selected"): 모든 콤보박스의 선택된 요소들을 선택
	- $(":checked")	: 체크된 모든 체크박스와 라디오버튼 요소들을 선택

 5-3) 기타 선택자
	- $(":header")	: 모든 헤더요소들(h1~h6) 선택
	- $(":animated"): 현재 애니메이션(animate() 함수)이 적용된 모든 요소
	- $(":focus")	: 현재 포커스가 있는 요소
	- $(":contains"): 지정된 문자열을 포함하는 요소
	- $(":empty")	: 비어있는 요소
	- $(":hidden")	: 숨겨진 요소 - type="hidden"과 넓이와 높이가 0으로 지정된 요소들
	- $(":visible")	: 현재 보이는 요소
	- $(":root")	: 문서의 루트요소
