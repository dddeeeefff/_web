jQuery 이벤트 메소드
 - 선택된 요소에 대한 처리를 함수에 연결시키는 기능
 - 문법
	$(선택자).이벤트메소드( 실행문; ) or $(선택자).이벤트메소드(function() { 실행문; });
	두 방식의 차이는 거의 없으나 주로 두번째 방법을 많이 사용함
 - 자바스크립트에서는 특정 요소에 이벤트 속성을 입력한 후 이벤트 발생시 특정 함수르 호출하는 방식으로 동작하나, jQuery에서는 요소에 이벤트 속성을 사용하지 않고 이벤트 리스너 개념으로 이벤트 발생을 캐치하여 작업함
	이벤트 리스너 : 컨트롤들에서 이벤트가 발생하는 지 여부를 감시하는 기능

1. 마우스 이벤트 메소드
 - click : 선택한 요소를 클릭할 때
 - dblclick : 선택한 요소를 더블 클릭할 때
 - mouseenter : 선택한 요소 안으로 마우스를 가져갈 때
 - mouseleave : 선택한 요소 안으로 마우스가 벗어날 때
 - mouseover : 마우스 포인터가 선택한 요소 위에 있을 때(mouseenter와 동일)
 - mouseout : 마우스 포인터가 선택한 요소 위를 벗어날 때(mouseleave와 동일)
 - mousedown : 선택한 요소를 마우스 버튼으로 누를 때
 - mouseup : 선택한 요소를 마우스 버튼으로 눌렀다가 뗄 때
 - mousemove : 선택한 요소 내에서 마우스가 움직일 때

2. 폼 이벤트 메소드 : 폼에 관련된 이벤트들로 주로 컨트롤에 대한 메소드들
 - submit : 폼이 submit 될 때 발생하며 form 태그에서만 발생
 - change : 요소의 값이 변경될 때 - input, textarea, select 태그에서 발생
 - select : 텍스트필드에서 임의의 텍스트가 선택될 때 발생
 - focus : 요소가 포커스를 얻을 때 발생
 - blur : 요소가 포커스를 잃을 때 발생
 - fousin : 자식요소가 포커스를 받으면 부모 요소에서 자동 실행
 - fousout : 자식요소가 포커스를 잃으면 부모 요소에서 자동 실행

3. 키보드 이벤트 메소드
 - keydown : 키를 누를 때 발생
 - keypress : keydown과 동일(일부 키 - alt, ctrl, shift, esc 에서는 동작 안함)
 - keyup : 키가 눌렸다가 올라올 때 발생
 - 이벤트 발생 순서 : keydown -> keypress -> keyup

4. 브라우저 이벤트 메소드
 - resize : 창의 크기를 변경할 때 발생
 - scroll : 지정된 요소에서 스크롤 할 때 발생

5. 바인딩 이벤트 메소드
 - on : 선택한 요소에 여러 가지의 이벤트 발생시 여러 개의 이벤트 처리기(핸들러)를 연결(여러 이벤트를 한 번에 처리할 수 있음)
 - off : on 메소드로 첨부된 이벤트 핸들러 제거
 - one : 선택한 요소에 여러 개의 이벤트 처리기를 연결 처리 함수가 한 번만 실행됨
 - trigger : 선택한 요소에 바인딩된 모든 이벤트를 트리거 - 강제 실행시키고 이벤트의 기본 기능도 실행함
 - triggerHandler : 선택한 요소에 바인딩된 모든 이벤트를 트리거 - 강제 실행, 기본 기능 안함

5-1. on(event [, data], function [, map])
 - event : 필수사항. 이벤트를 여러 개 지정(이벤트들은 공백으로 구분)
 - data : 선택사항. 함수에 전달할 추가 데이터를 지정
 - function : 필수사항. 이벤트가 발생할 때 동작할 실행 함수를 지정
 - map : 선택사항. 선택한 요소에 첨부할 하나 이상의 이벤트를 포함하는 이벤트 맵과 이벤트가 발생할 때 실행되는 함수
